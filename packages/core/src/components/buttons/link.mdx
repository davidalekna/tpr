# Link

To trigger an operation.

## When To Use

A link means an operation (or a series of operations). Clicking a link will trigger corresponding business logic.

## Usage

```js
import { Link } from '@tpr/core';
```

## Examples

### Basic usage

<Playground>
	<Link>Click to go back</Link>
</Playground>

### Usage with props

<Playground>
	<Link cfg={{ mr: 3 }}>Home</Link>
	<Link cfg={{ mr: 3 }}>Gallery</Link>
	<Link cfg={{ mr: 3 }} underline={true}>
		Log out
	</Link>
	<Link cfg={{ mr: 3 }} disabled>
		Disabled Link
	</Link>
</Playground>

### Usage with colours from the theme

<Playground>
	<Link cfg={{ mr: 3, color: 'danger.2' }}>Home</Link>
	<Link cfg={{ mr: 3, color: 'warning.1' }}>Gallery</Link>
	<Link cfg={{ mr: 3, color: 'success.1' }}>Log out</Link>
</Playground>

### Anchor tag

Using the anchorTag prop will create an `<a>` element, therefore the "disabled" prop will have no offect

<Playground>
	<Link
		cfg={{ mr: 3 }}
		anchorTag={true}
		href="http://www.google.com"
		target="_blank"
	>
		Home
	</Link>
</Playground>
