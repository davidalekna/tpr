## API

### Props

| Property           | Required | Type                        | Description                                                                                   |
| ------------------ | -------- | --------------------------- | --------------------------------------------------------------------------------------------- |
| complete           | true     | boolean                     | changes the state of the card when All details are correct                                    |
| i18n               | false    | IndependentTrusteeI18nProps | allows to override specific text in the card                                                  |
| independentTrustee | true     | IndependentTrustee          | the IndependentTrustee object                                                                 |
| onCorrect          | true     | function                    | actions for when the user clicks the "Confirm details are correct" checkbox                   |
| onRemove           | true     | function                    | actions for when the independentTrustee is removed                                            |
| onSaveAppointed    | true     | function                    | actions for when the user updates if the trustee was appointed to the scheme by the regulator |
| preValidatedData   | false    | boolean                     | allows to ignore the state of the "Confirm details are correct" checkbox & shows "No issues"  |
