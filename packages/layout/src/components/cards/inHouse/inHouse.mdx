# In House Admin Card

In House Admin Card for tpr apps

## Usage

```js
import { InHouseCard } from '@tpr/layout';
```

## Basic usage

<Playground>
	{() => {
		const [complete, setComplete] = useState(false);
		const callbackFn = (values, _fullInHouseAdminObject) => {
			console.log(values);
			console.log(_fullInHouseAdminObject);
			return Promise.resolve(values);
		};
		const inHouseAdmin = {
			id: '',
			schemeRoleId: 123,
			title: 'Mr',
			firstName: 'John',
			lastName: 'Smoth',
			effectiveDate: '1997-04-01T00:00:00',
			address: {
				addressLine1: 'Napier House',
				addressLine2: 'Trafalgar Pl',
				addressLine3: '',
				postTown: 'Brighton',
				postcode: 'BN1 4DW',
				county: 'West Sussex',
			},
			telephoneNumber: '01273 222 111',
			emailAddress: 'john.wick@warnerbros.com',
		};
		return (
			<InHouseCard
				addressAPI={{
					get: (endpont) => experianApiGet(endpont),
					limit: 100,
				}}
				complete={complete}
				inHouseAdmin={inHouseAdmin}
				onCorrect={(value) => setComplete(value)}
				onRemove={callbackFn}
				onSaveAddress={callbackFn}
				onSaveContacts={callbackFn}
				onSaveName={callbackFn}
			/>
		);
	}}
</Playground>

## API

### Props

| Property         | Required | Type                  | Description                                                                              |
| ---------------- | -------- | --------------------- | ---------------------------------------------------------------------------------------- |
| addressAPI       | true     | addressAPIType        | the required API params for the address search                                           |
| complete         | true     | boolean               | changes the state of the card when All details are correct                               |
| i18n             | false    | InHouseAdminI18nProps | allows to override specific text in the card                                             |
| onCorrect        | true     | function              | actions for when the user clicks the "Confirm details are correct" checkbox                  |
| onRemove         | true     | function              | actions for when the inHouseAdmin is removed                                             |
| onSaveAddress    | true     | function              | actions for when the user updates the inHouseAdmin's Address                             |
| onSaveContacts   | true     | function              | actions for when the user updates the Contact Details                                    |
| onSaveName       | true     | function              | actions for when the user updates the Name of the inHouseAdmin                           |
| prevalidatedData | false    | boolean               | allows to ignore the state of the "Confirm details are correct" checkbox & shows "No issues" |
| inHouseAdmin     | true     | InHouseAdminNoApi     | the inHouseAdmin object                                                                  |
