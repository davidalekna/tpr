# Checkbox

## Introduction

The `Checkbox` component should be used within the Form state provider.

| Props                  |
| ---------------------- |
| `checked`              |
| `onChange`             |
| `disabled`             |
| `align`                |
| `testId`               |
| `id`                   |
| `label`                |
| `value`                |

For a complete documentation on all the returned props, hook props and more
information check out the
[Github Page](https://github.com/davidalekna/tpr/tree/master/packages/forms#checkbox).

  {() => {
		const [checked, setChecked] = React.useState(false)
		return (
			<div style={{ width: 400, padding: 10 }}>
				<Checkbox
					checked={checked}
					onChange={() => {
