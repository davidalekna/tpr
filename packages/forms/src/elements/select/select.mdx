# Select

Select component to select value from options.

## When To Use

The select component should only be used as a last resort in public-facing services
because research shows that some users find selects very difficult to use.
See [Select - GOV.UK Design System](https://design-system.service.gov.uk/components/select/).

## Usage

`Inside your React project directory, run the following:`

```js
yarn add @tpr/forms
```

`or with npm`

```js
npm install @tpr/forms
```

`import items you wish to use from the library`

```js
import { FFSelect } from '@tpr/forms';
```

## API

### Props

| Property             | Required | Type     | Description                                                                            |
| -------------------- | -------- | -------- | -------------------------------------------------------------------------------------- |
| id                   | true     | boolean  | id assigned to the select element                                                      |
| testId               | false    | string   | data attribute for testers                                                             |
| ref                  | false    | ref      | A reference to the select element created using the useRef hook                        |
| label                | true     | string   | Label identifying the purpose of the field                                             |
| hint                 | false    | string   | Hint text to display after the label                                                   |
| options              | true     | array    | Options to select from                                                                 |
| required             | false    | boolean  | Sets whether the field is required. If not required an 'optional' label is displayed.  |
| optionalFieldLabel   | false    | string   | The label used to show the field is optional. Defaults to '(optional)'.                |
| onChange             | false    | function | Called when the selected value changes. Use ref.current.value to get the value.        |
| addPlaceholderOption | false    | boolean  | Adds an extra option with an empty value, above those provided by the options property |
| placeholder          | false    | string   | If addPlaceholderOption is true, the display text of the extra option                  |
| notFoundMessage      | false    | string   | Displayed as an option element when there are no valid options to select               |
| disabled             | false    | boolean  | Disable the select                                                                     |
| meta                 | false    | object   | Control when to display an error message                                               |
