# Number Input

Enter a number within certain range with the mouse or keyboard.

## When To Use

- When a numeric value needs to be provided

## Usage

`Inside your React project directory, run the following:`

```js
yarn add @tpr/forms
```

`or with npm`

```js
npm install @tpr/forms
```

`import items you wish to use from the library`

```js
import { FFInputNumber } from '@tpr/forms';
```

## Examples

[CodeSandbox](https://codesandbox.io)

<Playground>
	<Form onSubmit={(values) => console.log(values)}>
		{({ handleSubmit }) => (
			<form onSubmit={handleSubmit}>
				<FFInputNumber
					name="number_of_participants"
					label="Number of participants"
					hint="How many people are entering this event"
					validate={(value) => {
						if (value < 1 || value > 20 || !value || value === 0) {
							return 'Must be between 1 and 20';
						}
						return undefined;
					}}
					inputWidth={5}
					cfg={{ my: 5 }}
					after="cm"
					callback={(e) => console.log(e.target.value)}
				/>
				<button type="submit" style={{ display: 'none' }} children="Submit" />
			</form>
		)}
	</Form>
</Playground>

Example using record level validation

```js
import { Form, validate, renderFields } from '@tpr/forms';
```

<Playground>
	{() => {
		const fields = [
			{
				type: 'number',
				name: 'number_of_participants',
				label: 'Number of participants',
				hint: 'must be between 1 and 20',
				error: 'Cannot be empty',
				inputWidth: 5,
				cfg: { my: 5 },
			},
		];
		return (
			<Form onSubmit={console.log} validate={validate(fields)}>
				{({ handleSubmit }) => (
					<form onSubmit={handleSubmit}>
						{renderFields(fields)}
						<button
							type="submit"
							style={{ display: 'none' }}
							children="Submit"
						/>
					</form>
				)}
			</Form>
		);
	}}
</Playground>

## API

```
Accepted config props: FlexProps, SpaceProps
```

### Props

| Property | Required | Type    | Description                                  |
| -------- | -------- | ------- | -------------------------------------------- |
| cfg      | false    | object  | FlexProps & SpaceProps                       |
| disabled | false    | boolean | Disable checkbox                             |
| testId   | false    | string  | data attribute for testers                   |
| label    | true     | string  | Checkbox description                         |
| hint     | false    | string  | More detailed description about the checkbox |
