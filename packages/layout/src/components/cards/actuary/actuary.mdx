# Actuary Card

Actuary Card for tpr apps

## Usage

```js
import { ActuaryCard } from '@tpr/layout';
```

## API

### Props

| Property         | Required | Type             | Description                                                                                  |
| ---------------- | -------- | ---------------- | -------------------------------------------------------------------------------------------- |
| actuary          | true     | Actuary          | the Actuary object                                                                           |
| complete         | true     | boolean          | changes the state of the card when All details are correct                                   |
| i18n             | false    | ActuaryI18nProps | allows to override specific text in the card                                                 |
| onCorrect        | true     | function         | actions for when the user clicks the "Confirm details are correct" checkbox                  |
| onRemove         | true     | function         | actions for when the actuary is removed                                                      |
| onSaveContacts   | true     | function         | actions for when the user updates the Contact Details                                        |
| onSaveName       | true     | function         | actions for when the user updates the Name of the Actuary                                    |
| prevalidatedData | false    | boolean          | allows to ignore the state of the "Confirm details are correct" checkbox & shows "No issues" |
