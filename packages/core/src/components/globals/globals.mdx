# Flex

Flexbox building block with access to the theme variables

## When To Use

When you need to build a certain shape with padding, margin or background colour, set a direction of layout or separate 2 elements.

## Examples

### Basic usage

<Playground>
	<Flex
		cfg={{
			flex: '1 1 auto',
			justifyContent: 'space-between',
			bg: 'accents.2',
			color: 'white',
			p: 3,
		}}
	>
		<Flex>Left</Flex>
		<Flex>Right</Flex>
	</Flex>
</Playground>

## API

```
Accepted config props: FlexProps, SpaceProps, BackgroundProps, ColorProps, LayoutProps, CursorProps, TypographyProps
```
