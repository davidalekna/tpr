# Employer Card

Employer Card Card for tpr apps

## Usage

```js
import { EmployerCard } from '@tpr/layout';
```

## API

### Props

| Property         | Required | Type              | Description                                                                              |
| ---------------- | -------- | ----------------- | ---------------------------------------------------------------------------------------- |
| complete         | true     | boolean           | changes the state of the card when All details are correct                               |
| employer         | true     | Employer          | the Employer object                                                                      |
| i18n             | false    | EmployerI18nProps | allows to override specific text in the card                                             |
| onCorrect        | true     | function          | actions for when the user clicks the "Confirm details are correct" checkbox                  |
| onRemove         | true     | function          | actions for when the employer is removed                                                 |
| onSaveType       | true     | function          | actions for when the user updates the type of employer                                   |
| prevalidatedData | false    | boolean           | allows to ignore the state of the "Confirm details are correct" checkbox & shows "No issues" |
